# VDR-Suite Glossar

## Recording
Eine Aufnahme im VDR-System.

## Movie
Ein einzelner Film.

## Series
Eine Serie bestehend aus mehreren Episoden.

## Episode
Eine einzelne Serienfolge.

## Metadata
Beschreibende Informationen zu Medieninhalten.

## Artwork
Poster, Fanart, Banner und Vorschaubilder.

## Service
Logische Funktionsschicht der Suite.

## Module
Erweiterbare Funktionskomponente.

## Adapter
Verbindung zwischen externem Plugin und Suite.

## Provider
Externe Datenquelle oder Dienst.

## Job
Asynchron ausgeführte Aufgabe.

## Dashboard
Zentrale Startseite der Suite.
