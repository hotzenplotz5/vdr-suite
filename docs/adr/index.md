## ADR Numbering Policy

Canonical ADR sequence:

ADR-0001
ADR-0002
...
ADR-0016

Next available ADR:

ADR-0017

The historical lowercase adr-001 to adr-007 files remain for repository history and compatibility.

No new ADRs should be created in the lowercase series.
