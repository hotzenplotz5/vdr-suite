# VDR-Suite

VDR-Suite ist ein langfristiges Architekturprojekt zur Erweiterung des Video Disk Recorders (VDR).

Ziel ist die Bereitstellung gemeinsamer Services, Metadaten, APIs und moderner Oberflächen auf Basis des bestehenden VDR-Systems.

## Projektstatus

Aktuell befindet sich das Projekt in:

Phase 0 – Architektur und Planung

Noch keine produktive Implementierung.

## Langfristige Ziele

- Recording Service
- Metadata Service
- Artwork Service
- Job Service
- REST API
- modernes OSD
- WebUI
- Plugin-Integration

## Externe Projekte

Diese Projekte bleiben zunächst eigenständig:

- VDR-Rectools
- TVScraper

Die Integration erfolgt später über Adapter.
